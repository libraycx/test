
 思路：
    定义了程序所需的常量，包括注射剂量的最小值、最大值和递增步长，以及注射次数的最小值、最大值和递增步长，以及注射器型号的数量和注射剂量的数量。
    定义了一个包含注射剂量的数组，数组中的每个元素代表不同型号的注射器所能容纳的药液容量。
    使用两个嵌套的循环，外层循环遍历不同型号的注射器的药液容量，内层循环遍历不同的注射次数。
    在循环中，计算每种组合下的单次注射剂量，并输出结果。
